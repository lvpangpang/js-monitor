# js-monitor

js 监控 sdk

## 1. 安装

1.1 使用 cdn（推荐）

```js
<script src='https://lvpangpang.com/js-sdk-monitor/index.js'></script>
```

1.2 使用 npm 包

```js
yarn add js-sdk-monitor -D
```

## 2. 使用

```js
monitor.init({
  server: "你的服务端地址",
})
```

## 3.说明

这只是一个获取错误以及上传错误的 jssdk，对应的后端服务以及错误信息展示的管理后台需要你自己搭建
